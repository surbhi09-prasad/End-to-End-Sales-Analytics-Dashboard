# 📊 End-to-End Sales Analytics Dashboard

## Overview

Built an interactive Power BI dashboard to transform raw sales data into actionable business insights across products, retailers, regions, and sales channels.

## Business Problem

Organizations often rely on fragmented sales reports, making it difficult to track performance, identify trends, and support timely decision-making.

## Solution

Developed a 3-page Power BI solution featuring:

- Executive Summary Dashboard
- Product Drill-through Analysis
- Regional Performance Analysis

## Key KPIs

- 💰 Total Sales: $899.9M
- 📦 Units Sold: 2M
- 📈 Operating Profit: $332.13M

## Features

✅ KPI Cards

✅ Drill-through Navigation

✅ Dynamic Tooltips

✅ Month & Region Slicers

✅ MTD, YTD & MoM Analysis

✅ Geographic Analysis
